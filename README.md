# 1. midi_interface

MIDIデバイス（電子ピアノ・MIDIキーボード等）の入力を、Windowsのキーボード入力に変換するツール。

## 1. 概要
Windows Multimedia APIを使用してMIDI信号をリアルタイムにキャプチャし、指定された仮想キーボード操作へと変換。
これにより、楽器の演奏を通じてPC上のアプリケーションやゲームを操作することが可能。

## 2. システム構成
### 概要
**Event-Driven (イベント駆動型)** 処理によるMIDI信号のキャプチャ
効率的な信号変換を目的として設計。

### 特徴
* **コールバック制御**: `winmm.lib` の提供するコールバック関数により、信号受信時のみ処理が実行されるため、待機時のCPU負荷が低い。
* **仮想キーマッピング**: 内部の `switch-case` 構造により、MIDIノート番号からPCキーコードへの変換を計算量 $O(1)$ で実現。
* **ステータス解析**: ビット演算を用いてノートオン・ノートオフの状態、およびベロシティ（打鍵の強さ）を正確に判別。

## 3. 準備物
* **Windows PC**
* **MIDIキーボード または 電子ピアノ**（PCに認識されている必要あり）
* **USB-MIDIインターフェース**

## 4. 実行方法
本プログラムはWindows環境専用です。

### コンパイル
GCC（MinGW等）がインストールされている環境で、以下のコマンドを実行。（`winmm` ライブラリをリンクする必要あり）

```bash
gcc midi_interface.c -o midi_interface.exe -lwinmm
